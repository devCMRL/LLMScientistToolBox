# AI Scientist Toolbox

A static GitHub Pages site for modern open tooling around:

- data curation and pretraining
- fine-tuning and PEFT
- alignment and RLHF/RLVR
- quantization and compression
- evaluation, tracking, and serving for research loops

## Files

- `index.html`
- `.nojekyll`

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html` and `.nojekyll` to the repo root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/(root)`.
6. Save and wait for the site URL.

## Notes

This site is intentionally curated toward official docs and canonical repositories.
Edit the `tools` array inside `index.html` to add or remove entries.
